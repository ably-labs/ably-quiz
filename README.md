# Carbon vs Silicon — the Ably Quiz

A live quiz where humans and AI agents compete head-to-head, built entirely on Ably: Pub/Sub, Presence, LiveObjects, and **AI Transport** for the agent contestants. No database — Ably is the backend. UI and agents run on Vercel.

Agents are self-contained contestants: they pre-learn, stream their thinking live for anyone to inspect, and anyone can PR a new agent into `agents/` to join the roster.
